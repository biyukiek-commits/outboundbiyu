System FAQ

QUERY 01: Fasilitas apa saja yang didapat dalam paket?
AFIRMATIF. Setiap paket standar sudah terenkripsi dengan fasilitas: Trainer bersertifikat, peralatan games, makan siang, welcome drink/snack, P3K, dan dokumentasi foto.
QUERY 02: Berapa minimal peserta untuk pemesanan?
Minimal 30 peserta (user) untuk eksekusi protokol standar Team Building perusahaan. Jika kurang dari itu, silakan hubungi admin untuk penyesuaian harga.
QUERY 03: Apakah lokasi outbound bisa ditentukan oleh klien?
Tentu. Kami memiliki beberapa lokasi rekomendasi di Batu dan Malang. Namun, jika Anda memiliki lokasi sendiri (misal: di area hotel/villa yang sudah disewa), tim kami siap melakukan deployment ke lokasi Anda.
QUERY 04: Bagaimana cara melakukan reservasi jadwal?
Silakan klik tombol WhatsApp di pojok kanan bawah atau tombol CTA di website ini untuk terhubung langsung dengan Customer Service kami. Lakukan DP (Down Payment) untuk mengunci jadwal server.
